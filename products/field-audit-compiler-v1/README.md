# FOUNDER_FIELD_AUDIT_TO_WORKFLOW_COMPILER_v1 (sandbox authoring package)

A **Voice-to-Deterministic-Workflow Compiler**: the founder speaks defects while reviewing a real
surface → transcript → observations → issue clusters → deterministic workflow contracts → sandbox
jobs → verifier receipts → approval-ready patches → pattern expansion across similar routes/components.

**Authoring-only (authority=none).** No PASS is self-minted here; sandbox jobs only inspect / stage
diffs / test / emit receipts; canonical changes are approval-gated. Runners execute against a
founder-supplied `{TARGET_URL}` — the E2E example uses **fictional Acme placeholders**, not any live
surface. Infra identifiers are redacted to placeholders.

See `FIELD_AUDIT_COMPILER_DOCTRINE_v1.md` for the constitution and `FIRST_E2E_EXAMPLE_broken_pilot_v1.md`
for the full worked chain.
